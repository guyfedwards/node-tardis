# node-tardis

node implementation of [`tardis`](http://github.com/guyfedwards/tardis)

Creates .tar.gz of all folders in directory

### Installation
```
git clone git@github.com:guyfedwards/node-tardis
cd node-tardis
npm link
```

### Usage
```
tardis [dir]
```
